Notions

HTML

 Langage de balisage utilisé pour la création de pages web, permettant notamment de définir des liens hypertextes

CSS

 CSS est l'acronyme de « Cascading Style Sheets » ce qui signifie « feuille de style en cascade ». Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML).

PHP

 PHP (officiellement, ce sigle est un acronyme récursif pour PHP Hypertext Preprocessor ) est un langage de scripts généraliste et Open Source

Javascript

 JavaScript est un langage de programmation qui permet de créer du contenu mis à jour de façon dynamique
